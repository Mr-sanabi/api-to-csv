This project fetches data from a public API, processes JSON, and converts it into a structured CSV file.

Features:
- API data extraction (requests)
- JSON parsing
- Handling nested data (e.g. failures)
- Conversion to CSV

Technologies:
- Python
- requests
- json
- csv

Example use case:
- Extract SpaceX launch data
- Parse nested failure details
- Convert structured data into CSV for analysis

Example output included.
